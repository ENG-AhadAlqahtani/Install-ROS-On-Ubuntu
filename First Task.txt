*** Install Robot Operating System (ROS) On Ubuntu***
-First Step : downloaded the Virtual Box On the PC. 
-Seacond Step :  Installed the Ubuntu Operating System Version 16 on the Virtual Box. 
-Third Step : In the Commands interface (Terminal) I copied all the Commands required one by one to install the Robot Operating System. At The end ,the Rviz program opened on ROS and I was able to access the Robot emulator remotely.
